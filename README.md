
#### 1) What is the difference between `null` and `undefined`?

উত্তর ঃ undefined হলো  ভ্যারিয়েবল ডিক্লেয়ার করা হয়েছে কিন্তু কোন মান দেওয়া হয়নাই। Null মানে হলো ইচ্ছে করে খালি ভ্যালু দেওয়া হয়েছে।



#### 2) What is the use of the `map()` function in JavaScript? How is it different from `forEach()`?

উত্তরঃ map() function টা ব্যবহার করা হয় পুরাতন arry থেকে নতুন একটা array বানাতে। এই map() ফাংশন টা নতুন এরে রিটার্ন  করে কিন্তু পুরাতন array কোন পরিবর্তন করে না। যখন কোন ডাটা পরিবর্তন করে নতুন array দরকার হয় তখন এটা ব্যবহার করা হয়। অপরদিকে forEach() ফাংশন কোন array প্রতিটি উপাদান এর উপর লুপ চালায় কিন্তু কোন কিছু রিটার্ন করে না।

#### 3) What is the difference between `==` and `===`?

 উত্তরঃ `==` এটা শুধু মান এর কম্পেয়ার করে কিন্তু সেটা কোন  কোন টাপের ভ্যারিএবল সেটা দেখে না। কম্পেয়ার করার সময় ভিন্ন টাইপ পায় তাহলে convert করে পরে কম্পেয়া করে।কিন্তু অপর দিকে `===` ভ্যালু ও টাইপ ২টাই দেখে।যদি কোন টা false হল তাহলে সে false রিটার্ন করে দেয়।


#### 4) What is the significance of `async`/`await` in fetching API data?

 উত্তরঃ `async` api থেকে ডাটা এনে একটা প্রমিজ রিটার্ন করে । আর অপর দিকে await ব্যবহার করলে এই লাইনের কাজ শেষ না হওয়া পযন্ত সে অপেক্ষা করবে।
